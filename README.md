# robot-arm
The goal of this project was to create a robot arm that could identify and stack objects within its surrounding circular area. An Arduino Uno was used to control the electronics. The robot first completes a 360-degree scan of the surrounding area. The arm then proceeds to pick up and stack objects in order of decreasing size on top of the largest object. All objects within the surrounding arena and assumed to be cube shaped. I have developed a rudimentary program to verify the arm's functionality not shown here. However, the project shown here remains unfinished for now.
